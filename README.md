## Description

Repository for data and scripts needed for reproducing analysis on Master's Thesis. This repo will be updated with more information.

## Dependencies

This project uses [mise](https://mise.jdx.dev/) to handle python language versions and [uv](https://docs.astral.sh/uv/) for python packages and virtual environments. Once you have installed mise, the included `mise.toml` file will handle the setup of python and uv, cd into the directory and run:

```bash
mise install
```
